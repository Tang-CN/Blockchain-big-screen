# Blockchain-big-screen

大数据大屏项目

## 项目介绍

本项目是一个基于区块链技术的金融大数据大屏项目，旨在展示区块链技术在金融领域的应用和效果。项目采用了 Vue3.js 框架，结合 Echarts 图表库，实现了数据可视化展示。

## 功能特点

- 数据可视化：通过 Echarts 图表库，将区块链数据以图表形式展示，方便用户直观了解区块链技术的应用和效果。
